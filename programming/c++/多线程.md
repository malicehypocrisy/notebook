# 一、多线程基础

## &#9352;创建线程

* 通过创建`std::thread`类，创建一个新的线程并执行指定的函数。
* 线程的生命周期由`std::thread`对象的构造和销毁管理

## &#9353;线程分离

* `detach()`方法，用于将线程与主线程分离，使得线程独立执行。
* 分离后的线程无法通过 `join()` 等待其完成，因此主线程不再负责线程的生命周期。

# 二、线程同步与共享资源

## &#9352;互斥锁

* `std::mutex`：用于防止多个线程同时访问共享资源。
* `std::lock_guard<std::mutex>`确保对数据的访问呢是线程安全
